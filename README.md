# 🔗 SnipSnap

A sleek and simple URL shortener app built with **Next.js**, **Prisma**, and **SQLite**. Turn unwieldy URLs into crisp, shareable links in a single click.

---

## 🚀 Features

- 🔐 Unique, minimal short URLs
- 💾 SQLite for lightweight, file-based storage
- 🛠️ Prisma ORM for type-safe database access
- ⚡ Fast API routes with Next.js
- 🌐 Responsive UI built with Tailwind CSS

---

## 🧑‍💻 Tech Stack

- [Next.js](https://nextjs.org/) – React framework for server-rendered apps
- [Prisma](https://www.prisma.io/) – ORM for database access
- [SQLite](https://www.sqlite.org/) – Lightweight database
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
